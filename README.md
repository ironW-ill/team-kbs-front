# team-kbs-front

## eslint 관련
* console warning 처리
* semicolon 사용안함
* js 문법은 double("")이 아닌 single quote('')
* 탭(Tab) 사이즈는 space 2

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
