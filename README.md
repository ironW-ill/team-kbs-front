# team-kbs-front

## eslint 관련
* console warning 처리
* semicolon 사용안함
* js 문법은 double("")이 아닌 single quote('')
* 탭(Tab) 사이즈는 space 2

## firebase 배포 순서
* firebase CLI 설치 > npm install -g firebase-tools
* 로그인 > firebase login:ci
* firebase deploy --token "토큰값"

```
npm install
```

```
npm run serve
```

### Lints and fixes files
